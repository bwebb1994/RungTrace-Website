FAQ & terminology

The practical details plant controls engineers usually want before connecting RungTrace to a system.

Product and compatibility

Which PLCs are supported?

RungTrace supports Allen-Bradley ControlLogix and CompactLogix controllers. Both families receive the same core features, including tag discovery, monitoring and writes, trending, Gantt timelines, logic viewing, Tag Trace, and backplane scanning.

Do I need RSLinx?

No. You can connect directly by entering a PLC IP address or selecting a saved favorite. If RSLinx is installed, RungTrace can optionally use its driver list to discover existing PLC endpoints.

What can I do with an L5X file?

An L5X is a Studio 5000 project export. RungTrace uses it to graphically display ladder logic and Function Block Diagrams, show Structured Text, browse AOIs, cross-reference tags, and build Tag Trace results.

Which logic views support live values?

Ladder logic supports live monitoring, bit toggles, and supported value edits. Function Block Diagrams support live monitoring. Structured Text is currently view-only; live Structured Text values are planned for a later release.

Diagnostics and performance

What is Tag Trace?

Tag Trace finds the logic that writes to a selected tag and follows that chain through routines and Add-On Instructions. It helps isolate the bit preventing a system or sequence from enabling. A traditional cross-reference tool is also included.

Is 50 ms sampling guaranteed?

No. It is the fastest selectable sample interval. Actual performance depends on tag count, network latency, PLC load, and display workload. A small number of tags on a clean local network can often achieve 50 ms; larger or remote workloads may sample more slowly.

How are accidental writes prevented?

A global Write Enable control allows writes to be disabled while monitoring and diagnosing a system.

Trial and licensing

Is there a free trial?

Yes. A 30-day trial will be available when RungTrace launches.

Does RungTrace require a continuous internet connection?

No. Internet access is required once for initial activation. After activation, RungTrace works fully offline.

Can I move my license to another computer?

Yes. Customers can deactivate and transfer their machine-locked license themselves.

What happens when a subscription expires?

The RungTrace application can no longer be run until the subscription is renewed.

Common Rockwell terms

L5X
An XML-based project export created by Studio 5000. It contains controller logic, tags, routines, comments, and other project information.
AOI
Add-On Instruction: reusable controller logic packaged as a custom instruction.
CIP
Common Industrial Protocol: the industrial communication protocol used by EtherNet/IP devices.
EtherNet/IP
The industrial Ethernet network used to communicate directly with supported Logix controllers.
FBD
Function Block Diagram: a graphical programming language that connects functional blocks to represent control logic.
