From symptom to source

Stop searching. Start tracing.

RungTrace turns the question “why isn’t this on?” into a short, visible path through your actual program logic.

01

Load your project

Import a Studio 5000 L5X export. Programs, routines, comments, tags, and AOIs are mapped automatically.

02

Connect live

Connect directly over EtherNet/IP or use an existing RSLinx device. Live values appear directly on the rung.

03

Trace the cause

Select a blocked output. RungTrace follows every dependency and isolates the condition holding it back.
