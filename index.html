From symptom to source

Stop searching. Start tracing.

When the call is "Motor M300 is not running," RungTrace takes you from a site-wide search to the exact blocking condition in minutes.

01

Search every controller

Enter PLC addresses, choose saved favorites, or use an optional RSLinx driver to discover and connect to multiple controllers at once. Then search controller- and program-level tags to find exactly where Motor M300 is controlled.

02

Watch it live

Add the relevant tags to one unified watch list, monitor them in real time, and safely write values back when troubleshooting calls for it.

03

Trace the cause

Select any tag on the rung. Tag Trace follows every write, even through Add-On Instructions, and reveals the bit holding the process back.
